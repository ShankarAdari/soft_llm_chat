This is a responsive LLM web application with a Python backend and a React frontend, focusing on a soft, human-friendly tone for the LLM's responses, and including customization options like themes and live background themes. 


**Blueprint or this project**

LLM integration with soft, human-friendly tone system prompt for conversational responses

Real-time chat interface with message history and streaming support

Multiple theme options (light/dark/custom) with theme switcher

Live animated background themes that users can select and customize

Responsive design optimized for mobile, tablet, and desktop

Message markdown rendering for formatted LLM responses

Chat history persistence in database for logged-in users
